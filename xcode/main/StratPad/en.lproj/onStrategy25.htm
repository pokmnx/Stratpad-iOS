Metrics and Targets
You'll want to track the progress of each objective. For each objective, identify one key metric, a target value and a target date.

This is easier for some objectives. For example, "Increased gross margins" would look like this:

  • metric: gross margin %
  • target value: 27%
  • target date: Dec 31, 2012

Metrics for other objectives such as "higher quality" may be trickier to select. You might choose an internal metric such as "error rate". Or you could choose an external metric such as "customer perception of quality".

Some objectives are quite difficult to determine such as "fully trained staff". The value in the exercise is in thinking it through and coming up with something that you find meaningful. One example might be "the percentage of staff that have taken all the training". The target value would be 100% by an appropriate target date.
